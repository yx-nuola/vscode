RRAM测试结果可视化分析工具需求文档
一、项目概述
1.1 项目背景
为满足RRAM（阻变存储器）测试结果的高效分析需求，需开发一款支持二维矩阵图可视化、多模式数据呈现及交互式分析的工具，s帮助用户直观识别存储单元的测试差异并定位问题, 上传离线文件，前端解析出数据，然后渲染成二维矩阵图，并支持鼠标悬停/点击事件，表格-图形联动定位，颜色编码差异可视化，支持自定义颜色映射，支持数据导入、处理、呈现、联动定位。
1.2 项目目标
构建一个支持128×1024规模二维矩阵图展示、多格式数据导入、颜色编码差异分析及表格-图形联动定位的RRAM测试分析工具，提升测试数据处理效率与问题排查能力。
二、功能需求
2.1 二维矩阵图展示
2.1.1 矩阵规模支持

   1.  支持128列（WL，字线）×1024行（BL，位线）的二维矩阵图呈现RRAM测试结果数据，总单元数131,072个。
   2. 确保64×64 cells（4096个单元）的数据能够在单个屏幕内完整显示，无需滚动即可查看关键区域, 支持上传文件覆盖之前的数据， 可能第一次上传解析的数据是 64x64，后续再上传覆盖，可能最后的128x1024，则只显示128×1024的矩阵图。

2.1.2 行列定义

    数据排列规则：以BL（位线）为行，WL（字线）为列，形成1024行×128列的矩阵结构

2.2 颜色编码与差异可视化
2.2.1 色阶自定义

    用户可指定电流范围对应的颜色，例如：
        0~5mA：橙色
        5~10mA：蓝色
        支持多区间颜色配置，每个区间可自定义RGB颜色值。
    颜色映射需实时生效，矩阵图中每个cell根据测试电流值自动填充对应颜色。

2.2.2 差异直观呈现

    通过颜色梯度直观显示不同cell的测试结果差异，异常cell（如Fail）需高亮显示（如红色），正常cell（Pass）显示为绿色（参考截图示例）。

2.3 交互式信息展示
2.3.1 鼠标悬停/点击事件

    当鼠标悬停或点击矩阵图中的某个cell时，弹出信息窗口显示该cell的详细参数：
        设置电压（Vset）
        复位电压（Vreset）
        测量电流（Imeas）
        其他扩展参数（如测试时间、温度等，需支持配置）。

2.3.2 信息窗口布局

    信息窗口需包含“Information”“Address”“Logical”等标签页，分别展示基本信息、物理地址和逻辑地址数据。

2.4 数据呈现模式
2.4.1 双模式支持

    图形模式：二维矩阵图可视化，支持缩放（Zoom）、还原操作，参考截图中的工具栏功能
    表格模式：以表格形式展示原始测试数据，列包含BL、WL、Vset、Vreset、Imeas等字段，行对应每个cell的测试结果。

2.4.2 联动定位

    图形模式与表格模式支持相互定位：
        点击图形中的cell，表格自动滚动到对应行并高亮；
        点击表格中的行，图形自动定位到对应cell并高亮显示。

2.5 数据导入与处理
2.5.1 支持文件格式

    暂定支持以下格式：
        STDF（Standard Test Data Format）
        TXT（Data log）
        自定义txt文件（需定义字段分隔符、列名映射规则）。

2.5.2 导入模式

    追加模式：多个文件导入时，数据累加生成完整的bitmap图，适用于分批次测试数据的合并分析。
    覆盖模式（默认）：新文件导入时覆盖原有数据，确保数据一致性。

三、非功能需求
3.1 性能需求

    矩阵图渲染时间：128×1024规模数据加载后，矩阵图渲染完成时间≤2秒。
    交互响应时间：鼠标悬停/点击cell的信息弹出延迟≤0.5秒。

3.2 兼容性

    操作系统：Windows 10/11（64位），后续可扩展Linux支持。
    分辨率：支持1920×1080及以上分辨率，确保64×64 cells单屏显示。

3.3 可扩展性

    支持后续新增数据格式（如CSV、Excel）的插件式扩展。
    颜色映射规则、参数显示字段等需支持配置文件修改，无需重新编译代码。

四、界面设计参考（基于截图）
4.1 主界面布局

    左侧区域：矩阵图缩略图（显示整体数据分布，红色Fail区域高亮）、信息面板（显示Total、Zoom、Date等元数据）。
    右侧区域：主矩阵图（支持滚动条查看全量数据）、工具栏（包含Zoom、Hand、Top等交互按钮）。

4.2 信息面板字段

    Total：总单元数
    Zoom：当前缩放比例
    Radix：数值进制（如Hexadecimal）
    Date：数据生成时间
    Definition：数据定义（如column/block结构）
    Phy X/Y Max、Logi X/Y Max：物理/逻辑坐标范围
    Mode：测试模式（如Fail Capture）
    Bit Size：位宽配置（如bit0、bit1）

五、验收标准

    成功加载128×1024测试数据，矩阵图完整显示且64×64 cells单屏可见。
    颜色映射功能正常，用户自定义电流范围与颜色对应准确，异常cell高亮显示。
    鼠标悬停/点击cell时，信息窗口正确显示参数，表格与图形模式联动定位准确。
    支持STDF、TXT等格式导入，追加/覆盖模式功能正常，多文件合并生成bitmap图无数据丢失。

4. 颜色映射（可能是全白色，可能是配置的颜色）
5. 点击格子事件， 允许传入回调函数执行自己的业务逻辑，比如回调中拿到单元格数据，有表格的，定位表格数据；或者可能展示信息到某个地方，或者传输到某个地方
6. 鼠标悬停，信息窗口正确显示参数，表格与图形模式联动定位准确。
7. 数据支持拼接，支持覆盖模式，用户可配置，默认是覆盖模式，也可以选择覆盖模式 追加模式。
